# First Orbit Alpha Scope

## Goal

Build a complete, presentable alpha model for Staria VT's creator website.

This version should answer one question:

> Does the site immediately feel like Staria's own quiet little galaxy?

## Included

- Full-screen atmospheric cosmic layout
- Central Staria identity core
- Orbiting Social Galaxy on desktop
- Platform icons centered inside planet bodies
- Independent orbit behavior
- Hover/focus affecting only the selected planet
- Platform focus panel
- Mobile constellation card layout
- Small featured signal card
- Minimal footer
- Reduced motion support
- Clean Vue/SCSS architecture

## Not Included Yet

- Multi-page routing
- CMS
- Blog/news system
- Stream schedule integration
- Merch grid
- Long biography section
- Heavy canvas effects
- Tailwind

## Alpha QA Checklist

- Desktop galaxy has enough spacing between planets.
- Hovering one planet does not freeze the full galaxy.
- Planets are keyboard focusable.
- Mobile layout is intentionally simplified.
- Text remains readable over the cosmic background.
- Reduced motion mode removes continuous orbiting.
- Links are easy to update from the data file.
